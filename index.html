Portfolio



Bicycles and Statistical Analysis



This project centered around the analysis of consumer renting behavior at a specific bike shop, leveraging Python for comprehensive data analysis. The primary focus was to understand how different weather conditions influence the renting patterns of consumers. Through the utilization of Python's data analysis libraries, the project involved examining rental data to identify trends, correlations, and potential insights related to weather variables. The analysis encompassed exploring the impact of factors such as temperature, precipitation, and wind speed on the frequency and duration of bike rentals. By employing statistical techniques and visualizations, the project aimed to provide valuable insights into consumer behavior, enabling the bike shop to make informed decisions related to inventory management, marketing strategies, and overall business optimization. In summary, this project employed Python as a powerful tool for data analysis to uncover patterns and correlations within rental data, shedding light on the nuanced relationship between weather conditions and consumer behavior at the bike shop. The outcomes of this analysis can serve as a valuable resource for data-driven decision-making and strategic planning for the business.

Home Loans and Predictive Modeling



This project centered on leveraging Python and its key coding libraries, including pandas, numpy, and matplotlib, to conduct a linear regression analysis aimed at predicting loan qualification for applicants. The primary objective was to build a predictive model that could assess an applicant's likelihood of loan approval based on relevant variables. The utilization of pandas facilitated efficient data manipulation and preprocessing, while numpy enabled mathematical operations critical for the regression analysis. The project also harnessed the visualization capabilities of matplotlib to illustrate key insights and trends in the data. The culmination of these efforts resulted in a linear regression model with a commendable accuracy score of 0.8067. This score indicated the model's effectiveness in capturing and predicting patterns within the data, providing valuable insights into the factors influencing loan qualification. The high accuracy score underscored the reliability of the model, making it a valuable tool for financial institutions in evaluating loan applications and enhancing decision-making processes. In essence, this project showcased the power of Python and its libraries in conducting sophisticated data analysis, with a specific focus on building a predictive model for loan qualification, offering a robust tool for financial institutions to streamline their lending processes.





Diamonds and Exploratory Data Analysis



This project involved performing exploratory data analysis (EDA) on a diamonds dataset using Python and its prominent data analysis libraries, including pandas, numpy, and seaborn. The primary goal was to gain valuable insights into the characteristics and patterns present in the dataset, ultimately facilitating a deeper understanding of diamond attributes. Pandas and numpy were employed for efficient data manipulation, cleaning, and preprocessing tasks, ensuring the dataset was ready for analysis. The project utilized seaborn, a powerful visualization library, to create insightful graphs and charts, aiding in the exploration of relationships between different features of the diamonds. Through the application of descriptive statistics and visualizations, the project revealed key information about the dataset, such as the distribution of carat weights, cut quality, color grades, and prices. Correlations between these attributes were examined, providing a comprehensive overview of how different factors interplayed in the dataset. In summary, this project showcased the capability of Python and its libraries in conducting thorough exploratory data analysis on a diamonds dataset. The insights gained from this analysis can prove invaluable for jewelers, researchers, and industry professionals seeking a nuanced understanding of the factors influencing diamond characteristics and pricing trends.
